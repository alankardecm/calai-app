# FitTransform App

A premium fitness application designed to help you transform your body in 12-16 weeks.

## Features

- **Dashboard**: Daily overview of your goals, workout, and meals.
- **Workouts**: Detailed Push/Pull/Legs + Full Body routine.
- **Nutrition**: Complete meal plan with macros and calorie targets.
- **Profile**: Track your stats and goals.

## Getting Started

1.  Install dependencies:
    ```bash
    npm install
    ```

2.  Run the development server:
    ```bash
    npm run dev
    ```

3.  Open your browser at `http://localhost:5173`

## Tech Stack

- React
- Vite
- Tailwind CSS (via custom CSS variables for premium look)
- Lucide React (Icons)
- React Router DOM
